The original Minecraft

The Java version of Minecraft is the original edition of the game, released in November 18, 2011 by Notch. Even though now it is developed and maintained by Mojang Studios a part of Xbox Game Studios, which in turn is part of Microsoft. How can I play

To play the Java version of Minecraft you need to download and install the game on your computer. You can do it through official Minecraft website or other options alternatives. You can also buy the package that includes the Java version and the Bedrock version (the unified edition of cross-platform game) for Windows.
